What is HTML

HTML is a markup language used to structure content on websites.

It is simple, backbone, and easy to learn. It is the foundation of web development and is used to create the structure and layout of web pages.

With HTML, you can create headings, paragraphs, lists, links, images, and more.

What is CSS

CSS is used to style and layout web pages.

It works together with HTML to make websites visually appealing.

CSS Grid is a powerful layout tool that allows you to create complex, responsive layouts with ease.

What is GitHub CLI

GitHub CLI brings GitHub to your terminal.

It makes it easier to work with GitHub from the command line, allowing you to create repositories, manage issues, and more.
